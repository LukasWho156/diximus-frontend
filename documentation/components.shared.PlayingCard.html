PlayingCard

components.shared. PlayingCard

An animatable representaion of a playing card that can be interacted with. Animation is automatically done by React Motion whenever the card's position / scale parameters are changed.

new PlayingCard()

Properties:
Name Type Description
x number

the x coordinate of the card's top left corner, relative to the container
y number

the y coordinate of the card's top left corner, relative to the container
scale number

the relative scale of the card. 1 corresponds to a width of 1220 pixels and a height of 820 pixels.
angle number

the angle along the card's z axis
flip number

1 if the card's backside should be visible, 0 if the frontside should be visible
